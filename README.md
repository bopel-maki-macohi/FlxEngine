# FlxEngine

HaxeFlixel as a game engine with a UI n stuff this time, like a Unity or Godot, minus the 3D stuff.

## Credits

- Maki : Programmer

## Compiling

To compile the engine, this is all you need:

- [Haxe](https://haxe.org/download/)
- [HMM (Haxe Module Manager)](https://lib.haxe.org/p/hmm/)

And then run (in the repo)

- `hmm install`

and you should be good.
